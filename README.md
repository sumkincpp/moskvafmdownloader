moskvafmdownloader
==================

Github fork of http://sourceforge.net/projects/moskva-fm/

Dependencies
------

Perl, if it is not out of the box.

Also, there should be the following binaries :
  * faad (`faad2` or `faad` linux package)
  * lame (`lame` package)
  * mp3cut (Ubuntu `poc-streamer` package)
